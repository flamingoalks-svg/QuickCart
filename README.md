# QuickCart — README
## Описание проекта
QuickCart — учебный проект интернет-магазина, разработанный в рамках
курсовой работы.
Приложение предоставляет функционал корзины, каталог товаров, авторизацию
пользователей, административную панель и работу с базой данных.
## Цели проекта
- Изучение основ веб‑разработки
- Закрепление навыков работы с Django
- Изучение структуры MVC/MVT
- Практика в работе с БД, моделями, ORM
## Установка и запуск проекта
### 1. Клонирование репозитория
```
git clone https://github.com/flamingoalks-svg/QuickCart.git
cd QuickCart
```
### 2. Создание виртуального окружения
```
python -m venv venv
source venv/bin/activate # Linux, Mac
venv\Scripts\activate # Windows
```
### 3. Установка зависимостей
```
pip install -r requirements.txt
```
### 4. Настройка переменных окружения
Создайте файл `.env` в корневой директории проекта:
```
DEBUG=True
SECRET_KEY=ваш_секретный_ключ
EMAIL_HOST=smtp.mail.ru
EMAIL_HOST_USER=ваш_email
EMAIL_HOST_PASSWORD=пароль_приложения
```
### 5. Применение миграций и старт данных
```
python manage.py migrate
python manage.py loaddata initial_data.json
# или
python populate_db.py
```
### 6. Запуск сервера
```
python manage.py runserver
```
## Использование проекта
После запуска приложение доступно по адресу:
http://127.0.0.1:8000/
Функционал:
- просмотр каталога товаров
- корзина и оформление заказа
- регистрация и авторизация
- панель администратора:
http://127.0.0.1:8000/admin
## Структура проекта
- `/shop` — модели, представления, URL
- `/templates` — HTML-шаблоны
- `/static` — CSS, JS, изображения
- `populate_db.py` — скрипт заполнения базы
- `manage.py` — точка входа Django
